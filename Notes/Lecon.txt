Pandas :

TOP 20 à maîtriser (essentielles en pratique)
Inspection et exploration
head() / tail() — voir rapidement les premières/dernières lignes

info() — résumé du DataFrame (types, valeurs nulles)

describe() — statistiques de base (moyenne, min, max, etc.)

shape / columns / dtypes — structure du DataFrame

Nettoyage des données
drop() — supprimer des lignes ou colonnes

dropna() / fillna() — gérer les valeurs manquantes

duplicated() / drop_duplicates() — trouver/supprimer les doublons

astype() — convertir les types de colonnes

replace() — remplacer certaines valeurs

Transformation
apply() — appliquer une fonction sur les colonnes/lignes

map() / applymap() — appliquer une fonction sur chaque élément

groupby() — regrouper les données pour faire des agrégations

agg() / aggregate() — appliquer des fonctions d’agrégation

merge() / join() / concat() — combiner plusieurs DataFrames

sort_values() / sort_index() — trier les données

Sélection / Filtrage
loc[] / iloc[] — accéder aux lignes/colonnes

query() — filtrage avec des conditions logiques

isin() — filtrer avec des valeurs spécifiques

Bonus (à connaître selon les cas)
pivot() / pivot_table() — réorganiser les données

melt() — faire passer les colonnes en lignes (inverse de pivot)

value_counts() — compter les occurrences uniques

corr() — corrélation entre colonnes numériques

reset_index() / set_index() — gérer les index
